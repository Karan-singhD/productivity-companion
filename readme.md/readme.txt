# Productivity Companion

Productivity companion is a full stack web app that gives  productivity advice based on a user’s task or goal. Users enter a task and get concise AI  recommendations to help them stay focused and work more efficiently.

The application uses a React frontend and a Node.js and  Express backend. User prompts are sent to a RESTful API which works with Google Gemini AI to generate the suers  advice.

## Technologies Used
- React (Vite)
- Node.js
- Express
- Google Gemini API
- HTML, CSS, JavaScript

## Features
- AI-generated productivity recommendations
- REST API communication between frontend and backend
- Dynamic Gemini model detection
- Fallback advice system for reliability
